#Consumo de Veículos

Este projeto é um sistema em .NET que armazena dados sobre o consumo de combustível de veículos, utilizando SQL Server.

Projeto feito no modelo MVC, com frameworks Entity (ORM) e Identity (autenticação de usuários). Senhas de usuários criptografadas para armazenamento respeitando práticas seguras. CRUDs de veículos, consumos e usuários. Geração de relatórios.

Uma das telas pode ser vista em seu estado atual. O projeto se encontra em andamento.
